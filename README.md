## 앱 이미지 설명

    💥 주요 기능
  
      1. 첫번째 화면(테마 선택 화면)
  
         - 원하는 테마 선택
  

      2. 두번째 화면(유저 정보 입력 화면)

         -  이름, 단체명(팀명), 팀원 수 입력


      3. 세번째 화면(노래 재생 화면)

         -  노래 재생 후 정답 입력 버튼을 눌러 정답 입력 다이얼로그 생성
  
  
      4. 네번째 화면(정답 입력 화면)

         -  빈칸에 해당하는 정답 고른 후 다음 버튼을 눌러 다음 순서 인원에게 패스, 마지막 인원은 완료 눌러 종료


      5. 다섯번째 화면(노래 재생 화면)

         -  네번째 화면 완료 후 정답 확인, 다이얼로그에서 전달받은 정답자 목록 확인
         

      6. 여섯번째 화면(랭킹 화면)

         -  이름, 단체명 기준으로 한 테마에서 가장 많이 정답을 맞춘 유저 순서대로 나열
